# برومبت شامل لموقع الكابينيت

## 🎯 الهدف العام
إنشاء موقع ويب كامل (Full Stack) باسم **"الكابينيت"** باللغة العربية، بواجهة حديثة (Modern UI) وتصميم داكن (Dark Mode)، مخصص لمجموعة خاصة مكونة من **6 أعضاء فقط**.

---

## 👥 الأعضاء الستة (قائمة ثابتة)
1. **مؤنس الطويل**
2. **عبد الرحمن سارية الطويل**
3. **محمد العمصي**
4. **سالم أبو ستة**
5. **محمود المجايدة**
6. **محمد المجايدة**

---

## 🔐 نظام المصادقة والتسجيل

### نظام تسجيل الدخول
- **الخطوة 1**: عند دخول الموقع تظهر شاشة تسجيل دخول
- **حقل اسم المستخدم**: قائمة اختيار (Dropdown) تحتوي على الأعضاء الستة فقط
- **حقل كلمة السر**: حقل نصي لإدخال كلمة المرور

### نظام كلمة السر
- **أول دخول**: المستخدم لم يقم بتعيين كلمة سر من قبل
  - يُطلب منه إنشاء كلمة سر جديدة
  - يتم حفظ كلمة السر بشكل آمن (hashed)
- **الدخول اللاحق**: استخدام كلمة السر المحفوظة
- **تغيير كلمة السر**: يمكن تغيير كلمة السر من صفحة تعديل الملف الشخصي

### إعداد الحساب (أول دخول فقط)
بعد تعيين كلمة السر، يُطلب من المستخدم إدخال البيانات التالية (إلزامي):
- **الاسم الكامل**: نص حر
- **تاريخ الميلاد**: تاريخ (Date Picker)
- **صورة شخصية**: رفع صورة (Image Upload)
- **التخصص الدراسي**: نص حر
- **الهوايات**: نص حر

بعد إكمال البيانات، يتم الانتقال إلى الصفحة الرئيسية.

## 🏠 الصفحة الرئيسية (لوحة التحكم)

### التصميم العام
- **تصميم داكن حديث** (Dark Mode)
- **واجهة شبابية مرحة** مع لمسات كوميدية خفيفة
- **Responsive Design** - يعمل على الهاتف والويب

## ✨ الميزات الرئيسية (13 ميزة)

### 1️⃣ شات الاعتراف السري (الحنيوك) 💬
**الوصف:**
- منصة مشتركة بين الأعضاء الستة فقط
- يمكن لأي شخص إرسال رسالة لجميع الأعضاء بشكل **مجهول**
- جميع الرسائل تظهر في نفس الشات الجماعي

**الميزات:**
- اسم جميع المرسلين يظهر كـ: **"الحنيو..."** (مجهول)
- **إعادة صياغة بالفصحى**: الرسائل تُعاد صياغتها تلقائياً بالفصحى باستخدام الذكاء الاصطناعي
- إخفاء الهوية الكتابية للمرسل
- إشعارات عند وصول رسائل جديدة
- عرض الرسائل بترتيب زمني (الأحدث أولاً)

**التقنية:**
- استخدام LLM API لإعادة صياغة النصوص
- تخزين الرسائل الأصلية والمعاد صياغتها

---

### 2️⃣ حابب تعزم حد؟ (العزومات) 🎉
**الوصف:**
- نظام لإرسال واستقبال العزومات بين الأعضاء

**الميزات:**
- اختيار الشخص المراد عزمه من قائمة الأعضاء
- إدخال سبب العزومة (مناسبة، حفلة، إلخ)
- إرسال الطلب للشخص الآخر
- الشخص الآخر يستقبل إشعار بالعزومة
- خيارات الرد: **قبول** أو **رفض**
- عرض قائمة العزومات المعلقة والمقبولة والمرفوضة
- إشعارات عند قبول أو رفض العزومة

---

### 3️⃣ أفضل صورة الأسبوع 📸
**الوصف:**
- منصة لرفع الصور واختيار أفضل صورة كل أسبوع

**الميزات:**
- رفع الصور من الهاتف أو الويب
- عرض الصور في معرض (Gallery)
- نظام التصويت: كل عضو يمكنه التصويت لصورة واحدة فقط
- عرض عدد الأصوات لكل صورة
- اختيار الفائز تلقائياً (الصورة بأعلى أصوات)
- إعادة تعيين الأصوات كل أسبوع
- إشعارات عند إضافة صور جديدة

---

### 4️⃣ سجل الديون 💰
**الوصف:**
- نظام لتتبع الديون بين الأصدقاء

**الميزات:**
- **نظام الطلبات**:
  - الشخص الأول يختار الشخص الذي يريد منه المال
  - يدخل المبلغ والسبب
  - يرسل طلب للطرف الآخر
  - الطرف الآخر يستقبل إشعار بطلب الدين
  - خيارات الرد: **قبول** أو **رفض**
- **عرض التفاصيل**: تفاصيل الدين تظهر فقط للطرفين (المدين والدائن)
- **تسجيل السداد**: عند قبول الطلب، يمكن تسجيل السداد
- **سجل الديون المتبقية والمسددة**

---

### 5️⃣ المواقف المحرجة 😂
**الوصف:**
- أرشيف للمواقف المضحكة والمحرجة

**الميزات:**
- كل عضو يمكنه إضافة موقف محرج له
- **عرض اسم الناشر وصورته**: عند نشر موقف، يظهر اسم الناشر وصورته الشخصية
- عرض المواقف بترتيب زمني (الأحدث أولاً)
- إمكانية الإعجاب أو التعليق على المواقف (اختياري)

---

### 6️⃣ سجل الفوز بالبيس (PES) ⚽
**الوصف:**
- تسجيل نتائج لعبة PES يومياً

**الميزات:**
- تسجيل يومي للنتائج:
  - **الفائز**: من فاز باللعبة
  - **الخاسر**: من خسر
  - **من لم يلعب**: من لم يشارك
- عرض السجل الكامل بترتيب زمني
- إحصائيات: عدد الانتصارات والخسائر لكل لاعب
- إشعارات عند تسجيل نتيجة جديدة

---

### 7️⃣ دفتر التقييم ⭐
**الوصف:**
- نظام لتقييم الأعضاء من بعضهم البعض

**الميزات:**
- كل عضو يمكنه تقييم الأعضاء الآخرين (من 1 إلى 5 نجوم)
- إضافة تعليق اختياري مع التقييم
- عرض متوسط التقييم لكل شخص
- عرض عدد التقييمات لكل شخص
- عرض التقييمات الفردية (من قيّم ومتى)

---

### 8️⃣ صندوق النصائح المجهول 💡
**الوصف:**
- نظام لإرسال نصائح شخصية بشكل مجهول

**الميزات:**
- **اختيار الشخص المراد نصحه**: من قائمة الأعضاء
- **إرسال النصيحة**: مع اسم المرسل وصورته
- **خاص**: النصيحة تظهر فقط للشخص المستقبل
- **إشعارات**: إشعار عند استقبال نصيحة جديدة
- **عرض النصائح المستقبلة**: قائمة بجميع النصائح التي استقبلها المستخدم

---

### 9️⃣ أرشيف الصور 🖼️
**الوصف:**
- معرض لصور المجموعة

**الميزات:**
- رفع الصور من الهاتف أو الويب
- **عرض الناشر**: اسم الشخص الذي رفع الصورة
- عرض الصور في معرض (Gallery) مع إمكانية التكبير
- ترتيب الصور بالتاريخ (الأحدث أولاً)
- إمكانية البحث والتصفية (اختياري)
- إشعارات عند إضافة صور جديدة

---

### 🔟 الصدقة الجارية 🤲
**الوصف:**
- قسم روحاني مخصص لروح **محمد جمعة المجايدة**

**المحتوى:**
- **أدعية** باسم المتوفى
- **آيات من القرآن الكريم**

**الميزات:**
- **عرض اسم المرسل**: عند إضافة دعاء أو آية، يظهر اسم الشخص الذي أضافها
- **تصميم روحاني هادئ**: تصميم خاص وهادئ يعكس احترام المناسبة
- عرض الأدعية والآيات بترتيب زمني
- إمكانية الإعجاب أو التعليق (اختياري)

## 🎨 متطلبات التصميم

## 🔧 المتطلبات التقنية

### البنية التقنية
- **Frontend**: React 19 + Tailwind CSS 4
- **Backend**: Express.js + tRPC 11
- **قاعدة البيانات**: MySQL مع Drizzle ORM
- **المصادقة**: Manus OAuth
- **التخزين السحابي**: S3 لتخزين الصور
- **الذكاء الاصطناعي**: LLM API لإعادة صياغة النصوص

### الميزات التقنية
- ✅ نظام مستخدمين محمي (Authentication + Authorization)
- ✅ قاعدة بيانات MySQL
- ✅ رفع الصور وتخزينها في S3
- ✅ نظام إشعارات داخل الموقع
- ✅ تصميم Responsive
- ✅ استخدام API للذكاء الاصطناعي (إعادة صياغة النصوص)
- ✅ Real-time Updates (اختياري)
